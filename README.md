# ExtensaoCupom
Extensão de navegador simples para exibir cupons em vue com Api .NET 8. O objetivo deste projeto é apresentar experiência com arquitetura, autenticação JWT e fornecer uma base de como implementar uma extensão em vue.
O projeto possui camadas de Apresentação, Aplicação, Dominio, Dados e Configuração. Faz integração com banco de dados PostgreSql e utiliza "migrations" para criar e atualizar tabelas.
